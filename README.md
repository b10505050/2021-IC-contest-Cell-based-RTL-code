# 2021-IC-contest-Cell-based-RTL-code
2021 IC contest Cell based RTL code




請完成一地理圍籬系統，本系統使用 6 顆接收器在平面上建構出虛擬圍籬地理圍籬系統，依據接收器的座標及待測物體的座標來分析待測物體是在圍籬內或圍籬外。
